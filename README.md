# AxisApp

---

AxisApp is a simple multifunctional web application designed to combine multiple tools and features into a single unified interface.

## About the Project

AxisApp is a Flask-based web application that can include various modules and services (such as API integrations, user tools, information dashboard, etc.).

The project is mainly built for learning backend development with Flask and improving full-stack web development skills.

## Tech Stack

- Backend: Flask (Python)
- Frontend: HTML / CSS / JavaScript
- Template Engine: Jinja2
- API: REST / Fetch

## Features
- Modular Flask architecture
- Multiple utility modules
- Simple and clean UI
- Extendable structure for new features

## Installation

Clone the repository:

```bash
git clone https://github.com/adilkhanalimberdi/axisapp.git
cd axisapp
```


Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

#### Windows:
```bash
venv\Scripts\activate
```

#### Mac/Linux:
```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Flask server:
```bash
python app.py
```

#### Then open in browser: http://127.0.0.1:5000/
